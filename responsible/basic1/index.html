미디어쿼리(MediaQuery)

미디어 쿼리는 화면크기에 따른 각각의 속성값을 지정하여, 여러가지 화면을 구성하는 기술입니다.

@media only all and (조건문){실행문}

  • @media : 미디어쿼리가 시작됨을 표시함.
  • only : 미디어쿼리 구문을 해석하라는 명령어.(생략가능)
  • all : 미디어쿼리를 해석해야 할 대상을 나타냄.(생략가능)
    • all : 모든 미디어 유형에서 사용할 css를 정의함
    • print : 인쇄장치에서 사용할 css를 정의함
    • screen : 컴퓨터 스크린에서 사용할 css를 정의함
    • aural : 화면을 읽어 소리로 출려해 주는 장치에서 사용할 css를 정의함
    • tv : tv화면에서 사용할 css를 정의함
    • handheld : 손에 들고 다니는 장치에 사용할 css를 정의함
    • projection : 프로젝트를 위한 css를 정의함
  • and : 앞과 뒤의 조건을 나타냄(생략가능)
  • (조건문) : 해당 조건을 나타냄
  • {실행문} : 조건에 따른 실행을 설정함
